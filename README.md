The project is Intercity Banking Networking as it implements the
concept of banking system within the 3 sub branches and 1 main
branch. The branches are named as Hadeed, Malir and Saddar branch.
Each of the branch is connected via Router to Switch usi ng the concept
of Sub Interface Routing and having 2 Laptops in each branch with two
different networks.
The Routers connected with the Main branch use the concept of
Dynamic Routing and using the RIP protocol they are being
communicated with each branch. The Routers are connected via Fast
Ethernet and Serial ports.
Main Features:
Some of the main features of the project are;
Sub Interface Routing:
To communicate the traffic of different Vlans via Switch to Router and
using the single port. The concept is in Sub Branches Section.
Dynamic Routing:
To communicate the different Routers, as we use to communi cate the
Main and Sub Branches.
RIP Protocol:
This protocol is used to introduce the networks on the main router
which then allows the access to communicate.
